# SDN-Scale: AVL vs Red-Black

Projeto da disciplina de Estrutura de Dados II.

O objetivo é comparar duas estruturas de árvores balanceadas, AVL e Red-Black, em um cenário de roteamento de borda, simulando regras de pacotes com operações de inserção, busca e remoção.

## Integrantes

- Integrante 1 Gabriel Guarnieri Fraga e Sousa: Lead Software Engineer - implementação das estruturas AVL e Red-Black.
- Integrante 2: DevOps & SRE - testes de carga, coleta de métricas e geração de resultados.
- Integrante 3: QA & Analytics - validação das invariantes, code review e relatório técnico.
